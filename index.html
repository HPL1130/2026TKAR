Day 3：立山黑部阿爾卑斯山脈橫斷
1. 立山車站 ➔ 美女平【登山鐵道纜車】 1.7km / 7分 / 海拔977m 標高差約500公尺，可欣賞森林帶變化。此處有美女杉傳說及被譽為「日本百選」的山毛櫸原生林。
2. 美女平 ➔ 彌陀原 ➔ 室堂【高原巴士】 23km / 50分 / 海拔2450m 觀光中心地，室堂平可近距離眺望雄山、淨土山、別山「立山三山」的壯闊。
3. 室堂 ➔ 大觀峰【隧道電氣巴士】 3.7km / 10分 / 海拔2316m 穿越立山主峰下方。大觀峰是「移動的眺望台」，一享廣闊的空中散步。
4. 大觀峰 ➔ 黑部平【空中纜車】 1.7km / 7分 / 海拔1828m 長達1.7公里的無支柱直跨式纜車，長度為日本第一，视野極為寬廣。
5. 黑部平 ➔ 黑部湖【地下電車】 0.8km / 5分 / 海拔1455m 全線地下式設計，抵達攔截御前澤形成的翡翠綠人工湖。
6. 黑部湖 ➔ 黑部水壩【徒步】 0.8km / 15分 / 海拔1470m 漫步走過宏大的水壩上方，欣賞群山間隙中的碧綠水色。
7. 黑部水壩 ➔ 扇澤【電氣巴士】 6.1km / 16分 / 海拔1433m 乘坐巴士越過赤澤嶽隧道，抵達長野縣扇澤。
